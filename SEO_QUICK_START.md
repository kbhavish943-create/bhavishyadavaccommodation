# 🚀 QUICK START: SEO & GOOGLE SEARCH CONSOLE SETUP

## ⏱️ 5-MINUTE SETUP CHECKLIST

### Step 1: Submit to Google Search Console (2 min)
1. Go to: https://search.google.com/search-console/
2. Click "Add property"
3. Choose "URL prefix" method
4. Enter: `https://bhavishyadavlodge.com`
5. Verify ownership (DNS or HTML file method)
6. Once verified, continue to steps below

### Step 2: Submit Sitemap to GSC (1 min)
1. In Google Search Console, go to "Sitemaps"
2. Paste URL: `https://bhavishyadavlodge.com/sitemap.xml`
3. Click "Submit"
4. You should see "Success! Sitemap accepted"

### Step 3: Submit robots.txt (1 min)
1. Go to GSC → Coverage
2. This automatically checks robots.txt
3. Your robots.txt is already at: `https://bhavishyadavlodge.com/robots.txt`

### Step 4: Request Indexing (1 min)
1. In GSC, paste URL: `https://bhavishyadavlodge.com`
2. Click "Request Indexing"
3. Repeat for these key pages:
   - `https://bhavishyadavlodge.com/index.html`
   - `https://bhavishyadavlodge.com/seo-landing.html`
   - `https://bhavishyadavlodge.com/features.html`
   - `https://bhavishyadavlodge.com/ceremony-grounds.html`

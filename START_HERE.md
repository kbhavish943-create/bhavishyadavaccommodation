## 🚀 Next 30 Minutes - Get It Running

### Step 1: Get API Keys (10 minutes)

**Firebase:**
1. Go to https://firebase.google.com
2. Create new project
3. Copy credentials

**Stripe:**
1. Go to https://stripe.com
2. Get test API keys
3. Copy credentials

### Step 2: Install & Setup (10 minutes)

```powershell
# Install frontend
cd frontend
npm install

# Install backend
cd ../server
npm install

# Create .env file (copy .env.example)
copy .env.example .env
```

### Step 3: Run It (10 minutes)

**Terminal 1:**
```powershell
cd server
npm run dev
```

**Terminal 2:**
```powershell
cd frontend
npm run dev
```

Open http://localhost:3001 and you're done! 🎉

                      💡 AMOUNT CONVERSION

┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│  IMPORTANT: Razorpay uses PAISE for INR amounts                │
│                                                                  │
│  ₹1 = 100 paise                                                │
│  ₹10 = 1000 paise                                              │
│  ₹100 = 10000 paise                                            │
│  ₹150 = 15000 paise                                            │
│  ₹1000 = 100000 paise                                          │
│                                                                  │
│  Formula:                                                       │
│  paise = amount_in_rupees × 100                                │
│  rupees = paise ÷ 100                                          │
│                                                                  │
│  In Frontend:                                                   │
│  User enters: 150                                              │
│  We multiply: 150 × 100 = 15000                               │
│  We send: 15000 (paise)                                       │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
